### Food Delivery App :

## Develooped with :

 · Clean Architecture 
  
 · Dependency Injection  
  
 · State Management : CUBIT

 . Backend : Firebase


 ## App Features :

 . Favorite orders screen

 . Search screen

 . Stream Order Status screen ( Waiting - Preparing - Delivering )

 . Settings screen

  # Admin Section : 
                 . View All Orders
                 . Update Order Status
                 . Add New Meal
                 . Remove old meals
